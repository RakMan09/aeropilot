What is this?

AeroPilot is the flight-computer software for a model rocket — run entirely in simulation. A FreeRTOS firmware image executes on an emulated ARM Cortex-M3 (QEMU), "flies" a simulated launch, decides when to deploy the parachute, and radios telemetry down to this ground station. No physical hardware is involved.

This page replays the actual telemetry the firmware transmitted: each reading was packed into a CRC-protected frame, sent over a simulated ARINC-429 avionics bus, decoded here on the ground, and exported to JSON. Pick a flight and press Launch to watch it unfold.

What you're seeing

  • Altitude & vertical velocity estimated onboard by a Kalman filter
  • Apogee — the peak of the flight, where velocity crosses zero
  • Deploy — the parachute-release command
  • SAFE — the watchdog caught a hung task and failed safe

The side panel shows the live flight state, telemetry readouts, and bus-integrity stats (frames decoded, CRC/parity rejects). Switch to the Fault injection flight to watch the watchdog trip to SAFE.
